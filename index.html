How to view the HTML source of a web page in Chrome on a Mac:

"Inspecting Behind the Scenes"

  1. Open up the webpage you want to view in chrome. For an example we will open National Geographic
  2. Click the view tab which is located in the bar at the top of your computer next to "edit and "selection".
  3. Under view, ther should be a list. Click developer which islocated at the bottom of this list. It shoudl open up another list.
  4. Click "View Source" which should be the first element in this list. This should reload the page youre on and show the HTML source code.
  5. A shortcut for this is do click the keys "alt", "command" and "U" simultaneously.
  6. To get rid of the source code, delete the text "view-source:" from the URL at the top of chrome and reload the page.
